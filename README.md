## üöÄ Quick Start

Follow these steps to set up the project locally on your machine.

### Cloning the Repository

```bash
git clone https://github.com/AnuragSingh012/BookIt_Experiences_and_Slots.git
cd BookIt_Experiences_and_Slots
```

**Install dependencies**

```bash
npm install
```

**Run the development server**

```bash

```
